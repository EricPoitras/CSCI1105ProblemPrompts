Instructions

Write a program that returns the elements values at the diagonal of a square matrix as well as the sum of their values.

The element values of the square matrix will be given as a series of integers followed by -1 to indicate the last value.

You can assume that the square matrix has a maximum length of 4 rows and columns.

Example Test Case

Input

3
4
1
5
6
7
1
2
3
-1

Output

The sum of [3, 6, 3] is 12

Explanation

The diagonal of the matrix refers to the cell values at the intersection of the first column and row (3), the second column and row (6), the third (3), and so on.
